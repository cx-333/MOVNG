# Prerequisites

* python >= 3.x
* pytorch >= 1.1.x
* PyG 
* We provide the dependency file of our experimental environment, you can install 
all dependencies by creating a new anaconda virtual environment and running 
```shell
pip install -r requirements.txt
```
